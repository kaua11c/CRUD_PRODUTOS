CRUD de Produtos em PHP
Este projeto é um CRUD simples desenvolvido em PHP e MySQL para gerenciar produtos.

O sistema permite cadastrar, listar, editar e excluir produtos.

Funcionalidades:
  - Adicionar produto
  - Listar produtos
  - Editar produto
  - Excluir produto

Cada produto possui:
  - ID
  - Nome
  - Preço

Estrutura da Tabela: 
  CREATE TABLE produtos (
    id INT AUTO_INCREMENT PRIMARY KEY,
    nome_produto VARCHAR(100),
    preco DECIMAL(10,2)
  );

Tecnologias utilizadas:
  - PHP
  - MySQL
  - HTML
  - CSS

Objetivo: Projeto criado para praticar operações CRUD utilizando PHP e banco de dados MySQL.
